Instalación del Servidor HTTP

Apache HTTP

El servidor HTTP Apache es un servidor web HTTP de código abierto, para plataformas Unix (BSD, GNU/Linux, etc.), Microsoft Windows, Macintosh y otras, que implementa el protocolo HTTP/1.1 y la noción de sitio virtual según la normativa RFC 2616.

Instalación

Apache esta disponible en los repositorios oficiales así que se puede instalar de manera convencional

sudo apt install apache2

una vez terminada la instalación al dirigirnos a nuestra IP o dirección localhost podremos observar algo como esto.

Configuración del servidor

Los archivos de configuracion del servidor se encuentran en la carpeta

etc/apache2

En esta capeta se modificaran los archivos de apache2.conf, en el siguiente archivo buscamos la siguiente configuracion, la cual procederemos a sustituir.

#Configuracion Original

<Directory /var/www/html>

    Options Indexes FollowSymLinks

    AllowOverride None

    Require all granted

</Directory>

sustituimos por la siguiente linea incluyendo el directorio en en el que se ubica nuestra pagina.

#Configuracion para permitir .HTACCESS file

<Directory /var/www/test.com>

    Options Indexes FollowSymLinks

    AllowOverride All

    Require all granted

</Directory>

Restringir por nombre de usuario

Para la configuración de nombre de usuario es necesario agregar en nuestro archivo de .htaccess

#Authentificacion

AuthType Basic

AuthName "My Protected Area"

AuthUserFile /etc/apache2/.htpasswd

Require valid-user

Una ve teniendo la configuracion en el archivo httaccess es importante agregar las contrasenas a la ruta
/etc/apache2/.htpasswd
Para generar estas utilizaremos la siguiente herramienta
* http://www.htaccesstools.com/htpasswd-generator/ *

EX:
usuario: redes pass: redes

redes:s78Z/qZP$r6XM036bSqqPh0RRPzDDf.
